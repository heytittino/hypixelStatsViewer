# Hypixel Stats Viewer

A simple web application to view Minecraft player statistics from Hypixel.

## Features

- Search players by username
- View player statistics and achievements
- Clean, responsive interface

## Setup

### Prerequisites
- Python 3+ (for running the server)
- A Hypixel API key (get one from [here](https://developer.hypixel.net/))

### Installation

1. Clone or download this repository
2. Run the host.bat 
3. Navigate to `http://localhost:8000`

### Usage

1. Enter your Hypixel API key in the settings section
2. Enter a Minecraft username
3. Click "Search Player" to view stats
